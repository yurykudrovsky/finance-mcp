# Submission to awesome-mcp-servers

To submit `finance-mcp` to the [awesome-mcp-servers](https://github.com/punkpeye/awesome-mcp-servers) repository, follow these steps.

## Entry to Add

Locate the `Financial` or `Data` section in their `README.md` and insert this line:

```markdown
- [yurykudrovsky/finance-mcp](https://github.com/yurykudrovsky/finance-mcp) - Portfolio-grade MCP server for real-time stock quotes, historical OHLCV data, fundamentals, and technical indicators via yfinance. No API key required.
```

## Pull Request Description

**Title:** Add finance-mcp to Financial/Data section

**Description:**
```markdown
Hello! I'd like to submit my open-source MCP server to the awesome list.

**finance-mcp** allows Claude Desktop (and other clients) to easily query live and historical stock market data, compare assets, and compute technical indicators natively without requiring an API key (uses `yfinance` with an Alpha Vantage fallback).

Features:
- Live quotes and volume
- OHLCV historical fetching
- Financial fundamentals (P/E, Yield, Market Cap)
- Side-by-side asset comparison
- Technical Indicators (RSI, MACD, SMA)

Thank you for curating this list!
```
