# Security Policy

## Supported Versions

Currently, only the latest release of `finance-mcp` is supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability within `finance-mcp`, please do not open a public issue. Instead, send a private message or email to the repository owner directly. 

We take all security vulnerabilities seriously and will work to resolve them as quickly as possible. Please provide:

1. A description of the vulnerability.
2. Steps to reproduce the issue.
3. Any possible mitigations or solutions you might have found.

We will try to acknowledge your report within 48 hours and provide an estimated timeline for the fix.
